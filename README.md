# ДИКО

Интернет-магазин товаров для необычных питомцев.

## Запуск

```bash
npm install
npm run dev
```

Локальный адрес по умолчанию:

```text
http://127.0.0.1:5173/
```

## Сборка

```bash
npm run build
```

## Структура

- `src/App.tsx` - интерфейс и состояние приложения
- `src/data.ts` - каталог товаров, категории, животные и промо-слайды
- `src/styles.css` - стили приложения
- `index.html` - HTML-точка входа
