Интернет-магазин товаров для необычных питомцев.

## Возможности

- Авторизация, регистрация, выход и сохранение токена в браузере
- Личный кабинет с профилем, заказами, избранным, бонусами, промокодами и отзывами
- Каталог с поиском, сортировкой, фильтрами, сеткой/списком и подгрузкой товаров
- Карточка товара с галереей, похожими товарами, корзиной, избранным и отзывами
- Корзина, промокоды, оформление заказа и начисление бонусов
- Игра для получения бонусов и промокодов
- Светлая/тёмная тема и версия для слабовидящих
- Слайдер акций, недавно просмотренные товары, популярные товары, категории и бренды
- Дополнительно: сравнение товаров

## Запуск

```bash
npm install
npm run dev
```

Локальный адрес по умолчанию:

```text
http://127.0.0.1:5173/
```

## Сборка

```bash
npm run build
```

## Структура

- `src/App.tsx` - интерфейс и состояние приложения
- `src/data.ts` - каталог товаров, категории, животные и промо-слайды
- `src/styles.css` - стили приложения
- `index.html` - HTML-точка входа
